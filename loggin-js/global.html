formater

You can also use a custom formater if the default one does not satisfy your needs. In the formater you can access all log properties and you can also set the color of some segments of the log by using % folowed by one of: - r red - g green - b blue - p pink - y yellow - c cyan If you use %b lets say it will color until the breakpoint: [-,_|]
